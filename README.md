# Blinking LED using Arduino

## Objective
To blink an LED using Arduino Uno.

## Components
- Arduino Uno
- LED
- 220 Ohm Resistor
- Breadboard
- Jumper Wires

## Connections
- LED Anode -> Pin 13
- LED Cathode -> GND

## Working
The Arduino sends HIGH and LOW signals to the LED at intervals of one second, causing it to blink continuously.

## Output
The LED blinks every one second.

## Result
Successfully implemented LED blinking using Arduino Uno.
